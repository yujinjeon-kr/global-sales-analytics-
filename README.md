# Revenue Growth Quality Analysis

## 1. Problem
Revenue is growing, but is the growth sustainable?

## 2. Approach
- Decomposed revenue into:
  - Volume (orders)
  - Value (AOV)
- Applied YTD-aligned YoY comparison

## 3. Segmentation
- At Risk
- Volume-driven
- Healthy

## 4. Key Insight
Growth is largely volume-driven → declining monetization efficiency

## 5. Business Impact
- Pricing optimization required
- Risk markets need root-cause analysis

## 6. Tools
- SQL (data modeling)
- Tableau (dashboard)
